# AI Sign Language Learning & Assessment Platform

An AI-powered web application that helps users learn and assess sign language through real-time gesture recognition.

## Tech Stack

### Frontend
- React.js
- JavaScript
- Vite
- Tailwind CSS

### Backend
- FastAPI
- PostgreSQL

### AI/ML
- MediaPipe
- OpenCV
- TensorFlow
